## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/mokshith-portfolio.git
```

Go to the project folder

```bash
cd mokshith-portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```
